# Global Warming Explorer

A from-scratch rebuild of your original classroom presentation site — now a
13-page, mobile-friendly, interactive learning site under your own name,
built as a static HTML/CSS/JS project (no build tools, no framework — just
like your other GitHub Pages projects).

## What's inside

```
index.html                     Home
about.html                     About you + Techbase + your other projects
what-is-global-warming.html    Learn: the greenhouse effect
causes.html                    Learn: causes, ranked
effects.html                   Learn: global effects + Nigeria 2025 floods/heat
why-care.html                  Learn: why it matters, climate justice
carbon-footprint.html          Take Action: interactive footprint calculator
recycle.html                   Take Action: reduce/reuse/recycle
clean-energy.html              Take Action: solar, wind, Nigeria's opportunity
action-hub.html                Take Action: tickable checklist
data-explorer.html             Go further: charts (CO2, warmest years, renewables)
quiz.html                      Go further: 10-question interactive quiz
glossary.html                  Go further: A–Z glossary + sources
css/styles.css                 Design system (one file, all pages)
js/main.js                     Shared nav/scroll/counter behaviour
js/calculator.js               Carbon footprint calculator logic
js/checklist.js                Action Hub checklist logic
js/quiz.js                     Quiz logic + questions
js/charts.js                   Data Explorer charts (uses Chart.js via CDN)
images/profile/                Drop your photo here (see README.txt inside)
favicon.svg                    Browser-tab icon
build.py                       The generator script — edit this, not the .html files
```

## Adding your photo

Drop a photo at `images/profile/babatunde.jpg` — the About page will pick it
up automatically. Full details in `images/profile/README.txt`.

## How content gets edited

**Don't hand-edit the `.html` files directly** — they're generated by
`build.py`. To change any text, find the relevant block inside `build.py`
(it's organised page-by-page, clearly commented) and edit the text there,
then regenerate:

```
python3 build.py
```

This regenerates all 13 pages from the templates in one go, so your nav,
footer and design stay consistent everywhere automatically.

## Deploying to GitHub Pages (free, no card required)

1. Create a new repository on your GitHub account — e.g. `globalwarming` or
   `climate-explorer` — at github.com/babatundeawo, or under
   github.com/techbaseng if you'd rather it live under Techbase.
2. Upload everything in this folder (keeping the folder structure: `css/`,
   `js/`, `images/`, `favicon.svg`, all the `.html` files, `build.py`) to
   that repository.
3. In the repo, go to **Settings → Pages**, set **Source** to your main
   branch (root folder), and save.
4. GitHub will give you a live URL like
   `https://babatundeawo.github.io/climate-explorer/` within a few minutes.

## Images used

All photos are real, freely-licensed images pulled from Wikimedia Commons
(NASA public-domain Earth photo, and Wiki Loves Africa contest photos from
Nigeria), loaded directly from Commons — no copies needed, no copyright
concerns, and full credit lines are shown on each photo. Everything else
(the greenhouse-effect diagram, icons, charts) is built directly in the
site's own CSS/SVG/JS, so there's nothing else to license or attribute.

## Notes on the data

Every statistic on the site (CO2 ppm, global temperature anomalies,
renewables share, Nigeria's 2025 flood and heat figures) is sourced from
NASA, NOAA, the IEA, and Nigerian agency reporting (NiMet/NEMA) as of mid-
2026. The Glossary & Sources page links to all of them. Worth a light
refresh once a year or so as new annual figures come out — search for
"NOAA global climate report [year]" and "IEA global energy review [year]"
and swap the numbers in `causes.html`, `effects.html`, `clean-energy.html`,
`data-explorer.html` and the relevant readout strips inside `build.py`.
